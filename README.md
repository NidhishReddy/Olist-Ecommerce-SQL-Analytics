# Olist-Ecommerce-SQL-Analytics
Olist E-Commerce SQL Analytics
End-to-end analytics project on 100K+ real Brazilian e-commerce orders using Python and PostgreSQL.
Topics covered: Star Schema Design, RFM Segmentation, Cohort Retention, Seller Scorecard, Churn Detection, Delivery Impact Analysis and Business Scenario Modelling.
Dataset: Olist Brazilian E-Commerce Public Dataset (Kaggle)
Tools: Python, PostgreSQL, Pandas, Matplotlib, Seaborn, SQLAlchemy
